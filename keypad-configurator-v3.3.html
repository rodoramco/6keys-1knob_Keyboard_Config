🖱 Macro Helper (AHK)
Mouse clicks and wheel events can't be emitted by the dongle directly, so this app generates an AutoHotkey v2 script that listens for hidden trigger combos (Ctrl+Alt+Shift+F13…F21) and plays back the macro steps you defined. Install AHK v2, run the .ahk file, keep it running while you use the keypad.
